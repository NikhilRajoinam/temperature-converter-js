# temperature-converter-js

A simple temperature conversion tool built using HTML and JavaScript.

## Features
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Uses radio buttons for unit selection
- Displays the converted value instantly

## Technologies Used
- HTML
- JavaScript

## How It Works
1. Enter a temperature value
2. Select the conversion type
3. Click the submit button
4. The converted temperature is displayed on the screen

## Learning Outcomes
- DOM manipulation
- Handling user input
- Conditional logic in JavaScript
